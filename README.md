# Airline Loyalty Program — End-to-End DE & BI Project

##  Project Overview
End-to-End Data Engineering & Business Intelligence solution 
for Northern Lights Air (NLA) — a fictional Canadian airline.

Built as part of **NTI Digital Egypt Youth Program**  
**Data Engineering & Analytics Track — 2026**

##  Dataset
| File | Rows | Columns |
|------|------|---------|
| Customer Loyalty History | 16,737 | 16 |
| Customer Flight Activity | 392,936 | 8 |
| Calendar | 2,557 | 4 |

##  Tech Stack
- **SSIS** — 3-layer ETL: ODS → Staging → Data Warehouse  
- **SQL Server** — Galaxy Schema (2 Facts, 4 Dimensions)  
- **Power BI** — 3 Interactive Dashboards  

##  Key Findings
- 2018 promotion added 971 members but minimal flight impact
- 508,808 total flights | 762M km | $2.21M revenue from redemptions
- Real loyalty driven by standard long-term members
